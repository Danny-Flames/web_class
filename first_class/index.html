CSS

Three methods of styling

  • inline styling
  • Internal styling
  • External styling

Mrgin & Padding

Padding is the space between the element and its content.

Margin is the space outside of the element and it's parent element

Inline styling is done in-between the opening tag of the element(note that this element may be h1, h2, div, p and so on...).

Internal styling is done in between the opening head element(usually found at the top of the html document) and the closing head element. And internal styling is done inside the style element. In summary, the internal styling will be inside the style element and the style element will be inside the opening and closing head tags.
